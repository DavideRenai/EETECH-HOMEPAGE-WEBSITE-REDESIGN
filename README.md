# ☁️ Cloud Services Homepage — Redesign Case Study

A UX/UI redesign case study of a cloud services provider homepage, from research and design in Figma to a fully coded implementation in HTML & CSS.

---

## 📌 Overview

This project documents the end-to-end redesign process of a cloud services homepage. The goal was to improve visual clarity, user experience, and overall aesthetic while maintaining the brand's identity. The result includes both a detailed case study and a working front-end prototype.

---

## 📁 Repository Structure

```
├── index.html         # HTML structure of the redesigned homepage
├── styles.css         # CSS styling and layout
├── case-study.pdf     # Full case study (exported from Figma)
└── case-study.svg     # Case study in vector format
```

---

## 🎨 Design Process

The case study covers the full design workflow:

- **Research** — Analysis of the original site's pain points and UX issues
- **Wireframing** — Low-fidelity sketches and layout exploration
- **UI Design** — High-fidelity mockups built in Figma

📄 The full case study is available in [`case_study.pdf`](./case_study.pdf) or [`case_study.svg`](./case_study.svg)

---

## 💻 Development

The design was translated into a static front-end using:

- **HTML5** — Semantic markup and page structure
- **CSS3** — Custom styling, layout (Flexbox & Grid), responsive design and dark mode

### Run locally

No dependencies or build tools required. Just open the file in your browser:

```bash
open index.html
```

Or simply double-click `index.html` in your file explorer.
